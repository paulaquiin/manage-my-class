Instalación con Render

- Acceder al siguiente enlace: https://manage-my-class.onrender.com/registro/
- Esperar a que Render inicie el servicio y acceder como usuario



Instalación mediante GitHub

- Acceder al repositorio del proyecto en Github
- Pulsar en “< > Code” → HTTPS → copiar la url (también puedes copiarlo directamente de aquí: https://github.com/paulaquiin/manage-my-class.git)
- Abrir una nueva ventana en Visual Studio Code. Pulsar en Terminal → New Terminal
- Copiar este comando para traer el proyecto:  git clone https://github.com/DevonCrawford/Video-Editing-Automation.git
- Recuerda que se copiará en la ruta en la que estés, puedes moverte a diferentes carpetas cuando el comando cd y el Tab.
- Para abrir el proyecto hay dos opciones: 
1. Buscar la carpeta que has creado en la ruta que tenías seleccionada desde “Open Folder” en el propio VSC
2. En la terminal → Moverte utilizando cd y Tab hasta estar en la carpeta del proyecto y escribir el comando code .
- Escribir en la terminal: node backend/server.js para iniciar el proyecto
- Acceder a la url en localhost que te responda la terminal

  
Instalación con el código descargado

- Descargar el archivo ZIP del código
- Extraer el archivo
- Abrir VSC u otro editor de código, y en el menú seleccionar “Open Folder” o cualquier opción de abrir carpeta
- Seleccionar la que se ha descomprimido antes
- Escribir en la terminal: node backend/server.js para iniciar el proyecto
- Acceder a la url en localhost que te responda la terminal
